# ゲーム別投稿一覧機能 - 使用方法

## 概要

Novel Game Plugin に新しく追加されたゲーム別投稿一覧機能により、各ゲームタイトルごとに投稿（シナリオやコマンド）をまとめて表示できるようになりました。

## 機能一覧

### 1. 管理画面での表示
- **ゲーム別一覧ページ**: 各ゲームタイトルに属する投稿を整理して表示
- **フィルタリング機能**: 投稿一覧画面でゲームタイトルによる絞り込み
- **投稿数の表示**: 各ゲームに属する投稿数を確認可能

### 2. フロントエンド表示
- **ショートコード**: `[noveltool_game_posts]` でページに表示
- **柔軟な表示オプション**: タイトル表示/非表示、投稿数制限など
- **レスポンシブ対応**: スマートフォンでも見やすい表示

### 3. 開発者向け機能
- **API関数**: プログラムから投稿を取得可能
- **フック**: カスタマイズ用のアクションとフィルター
- **国際化対応**: 翻訳可能なテキストドメイン

## 管理画面での使い方

### ゲーム別一覧ページ
1. WordPress管理画面にログイン
2. 「ノベルゲーム」メニューを選択
3. 「ゲーム別一覧」サブメニューをクリック
4. 各ゲームタイトルごとに投稿が整理されて表示されます

### フィルタリング機能
1. 「ノベルゲーム」→「ノベルゲーム一覧」を選択
2. 「すべてのゲーム」ドロップダウンから特定のゲームタイトルを選択
3. 選択したゲームに属する投稿のみが表示されます

## ショートコードの使い方

### 基本的な使い方

```
[noveltool_game_posts]
```
すべてのゲームの投稿を表示します。

### 特定のゲームの投稿のみ表示

```
[noveltool_game_posts game_title="マイゲーム"]
```

### タイトルを非表示にして表示

```
[noveltool_game_posts show_title="no"]
```

### 投稿数を制限して表示

```
[noveltool_game_posts limit="5"]
```

### 抜粋を表示

```
[noveltool_game_posts show_excerpt="yes"]
```

### 複数のオプションを組み合わせ

```
[noveltool_game_posts game_title="マイゲーム" show_title="yes" limit="3" show_excerpt="yes"]
```

## 開発者向けAPI

### 主要な関数

#### `noveltool_get_posts_by_game_title($game_title, $args)`
特定のゲームタイトルの投稿を取得します。

**パラメータ:**
- `$game_title` (string): ゲームタイトル
- `$args` (array): 追加のクエリパラメータ

**戻り値:**
- `array`: 投稿の配列

#### `noveltool_get_all_game_titles()`
すべてのゲームタイトルを取得します。

**戻り値:**
- `array`: ゲームタイトルの配列

#### `noveltool_get_posts_grouped_by_game()`
ゲームタイトル別に投稿をグループ化して取得します。

**戻り値:**
- `array`: ゲームタイトルをキーとした投稿配列

### 使用例

```php
// 特定のゲームの投稿を取得
$posts = noveltool_get_posts_by_game_title('マイゲーム');

// すべてのゲームタイトルを取得
$game_titles = noveltool_get_all_game_titles();

// ゲーム別にグループ化された投稿を取得
$grouped_posts = noveltool_get_posts_grouped_by_game();
```

## CSSカスタマイズ

フロントエンドの表示をカスタマイズするには、以下のCSSクラスを使用できます：

```css
.noveltool-game-posts {
    /* ゲーム別投稿コンテナ */
}

.noveltool-game-title {
    /* ゲームタイトル */
}

.noveltool-posts-list {
    /* 投稿リスト */
}

.noveltool-posts-list li {
    /* 投稿アイテム */
}

.noveltool-posts-list a {
    /* 投稿リンク */
}

.noveltool-post-excerpt {
    /* 投稿抜粋 */
}
```

## 国際化（翻訳）

このプラグインは翻訳可能です。テキストドメインは `noveltool` です。

翻訳ファイルは `languages/` フォルダに配置してください。

## トラブルシューティング

### 投稿が表示されない
- 投稿にゲームタイトルが設定されているか確認してください
- 投稿のステータスが「公開」になっているか確認してください

### フィルタリングが動作しない
- WordPressの管理画面で「ノベルゲーム」の投稿一覧を開いているか確認してください
- ゲームタイトルが正しく設定されているか確認してください

### ショートコードが表示されない
- ショートコードが正しく記述されているか確認してください
- プラグインが有効化されているか確認してください
