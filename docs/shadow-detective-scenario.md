# 「影の探偵」本格推理ゲーム シナリオ詳細設計

## ゲーム概要

**タイトル:** 影の探偵（Shadow Detective）  
**ジャンル:** 本格推理ノベルゲーム  
**シーン数:** 23シーン  
**エンディング数:** 4種類（完全解決・部分解決・誤推理・証拠不足）

## ストーリー概要

著名な実業家・黒崎誠の失踪事件を追う探偵のストーリー。
表向きは単純な失踪事件だが、調査を進めるにつれて闇取引・裏社会との繋がり・家族の秘密が明らかになっていく。
プレイヤーの選択と証拠収集によって、真相解明の度合いが変化する。

## アイテム・フラグ定義

### 証拠アイテム（5種）

| フラグID | アイテム名 | 取得シーン | 用途 |
|---------|-----------|----------|------|
| `flag_item_watch` | 懐中時計 | シーン3 | 失踪者の所有物、刻印がヒントに |
| `flag_item_note` | 失踪者の手記 | シーン8 | 失踪者の心情と取引の証拠 |
| `flag_item_photo` | 証拠写真 | シーン6 | 裏取引現場の写真 |
| `flag_item_key` | 隠し部屋の鍵 | シーン10 | 隠し部屋へのアクセスキー |
| `flag_item_trade_memo` | 闇取引メモ | シーン14 | 黒幕の正体を示す決定的証拠 |

### 調査進捗フラグ

| フラグID | 説明 | 設定タイミング |
|---------|------|--------------|
| `flag_talked_wife` | 妻と会話済み | シーン5 |
| `flag_talked_friend` | 友人と会話済み | シーン6 |
| `flag_found_hidden_room` | 隠し部屋発見 | シーン10 |
| `flag_met_underworld` | 裏社会接触 | シーン12 |
| `flag_confronted_mastermind` | 黒幕対峙 | シーン18 |

## エンディング分岐条件まとめ

### 1. 完全解決エンド（シーン21）

**必須条件（AND）:**
- `flag_item_watch: 1`
- `flag_item_note: 1`
- `flag_item_photo: 1`
- `flag_item_key: 1`
- `flag_item_trade_memo: 1`
- `flag_confronted_mastermind: 1`
- シーン20に到達

### 2. 部分解決エンド（シーン22）

**条件:**
- シーン20に到達
- 上記完全解決の条件を満たさない
- 最低限の証拠（flag_item_watch, flag_item_photo, flag_item_note）のうち2つ以上

### 3. 証拠不足・誤推理エンド（シーン23）

**条件:**
- `flag_item_key: 0`（隠し部屋の金庫を開けられなかった）
- または、シーン17で証拠なしで引き下がった
- または、シーン20に到達できなかった

### 4. ゲームオーバー（シーン23に同じ）

**条件:**
- 重要な選択肢で誤った判断
- 必須シーン（シーン3, 8, 14）をスキップした

## 実装上の注意事項

1. **国際化対応:** すべてのテキストは `__()` 関数でラップし、textdomain `novel-game-plugin` を使用
2. **フラグ管理:** localStorage を使用したクライアントサイドでのフラグ保存
3. **条件分岐:** フロントエンドJavaScript で条件チェックを実装
4. **エンディング判定:** シーン20でフラグ状態を総合的に評価
5. **プレースホルダー画像:** SVG形式の簡易画像を使用（本実装では実際の画像に置き換え可能）
